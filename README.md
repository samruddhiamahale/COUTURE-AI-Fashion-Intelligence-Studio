# ✦ COUTURE AI — Fashion Intelligence Studio

An industry-level AI-powered fashion platform built with Streamlit.

## 🚀 Features
- AI Fashion Design Generator (Gemini API)
- Trend Forecasting Dashboard
- Outfit Builder with Personalization
- Cost Intelligence System
- AI Image Generation (Stable Diffusion)
- Design History Tracking

## 🧠 Tech Stack
- Python
- Streamlit
- Google Gemini API
- Hugging Face (Stable Diffusion)
- Matplotlib / NumPy

## ⚙️ Setup Instructions

1. Clone repo: git clone https://github.com/samruddhiamahale/couture-ai.git

cd couture-ai

2. Install dependencies: pip install -r requirements.txt

3. 3. Create `.env` file: GEMINI_API_KEY=your_key_here
HF_API_KEY=your_key_here

4. Run app:

streamlit run ai_fashion.py
